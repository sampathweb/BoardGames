===================
Capitalize Package
===================

A Package that defines the Rules of Chess.  You can play chess as a Two Player Game or Play against the Computer (Pacakge).  To play against the computer the package needs a database engine with pre-loaded games to draw intelligence from.

Purpose
========

This package can be used by a Web App or another Applciation to play chess.  Applications can provide a database connection to the chess engine to gain knowledge from grandmaster games.
